First off, did you know that the web and the internet are two different things?! What you refer to as the 'internet' is actually the web! See here.

The navigator object can obtain information about the current browser the user is using. Here we'll list out six properties you should be aware of as you browse the web.

  • "geolocation" - this can access the user's current location and user permissions settings can deny or allow access of that information.
  • "appName" - this displays the name of the web browser accessing the page.
  • "onLine" - whether the browser currently has a network connection.
  • "userAgent" - this is a string storedin the HTTP user-agent handler request; it contains information about the browser, platform name, and compatibility.

The screen object is used to obtain information about the display screen's size, resolution, and color depth.

  • "height" - the height of the display screen.
  • "pixelDepth" - display screen's color resolution in bits per pixel.
